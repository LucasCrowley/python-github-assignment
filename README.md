# Study Time Tracker

This project is a simple Python program that asks the user how many hours they studied today, checks for invalid input, and calculates an estimate of weekly study hours based on their response. It helps users stay aware of their study habits and track progress in an easy and interactive way.

## How to Run the Program

Make sure you have Python installed.

Open a terminal or command prompt in this project’s folder and run:

python app.py

(If your system uses Python 3 specifically, run:)

python3 app.py

## Example Output

Welcome to the Study Time Tracker!
How many hours did you study today? 2
You studied 2.0 hours today.
At this pace, you will study about 14.0 hours this week!
